_idris-bash_
============

GNU _bash_ backend for Idris.

Based on Edwin Brady’s [PHP backend](https://github.com/edwinb/idris-php).

**Not functional.  Work in progress.**


Example
-------

    $ idris pythag.idr --codegen bash -o pythag.sh
    $ bash pythag.sh
    ...
